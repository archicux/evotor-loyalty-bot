ИНСТРУКЦИЯ ПО УСТАНОВКЕ:

1. СЕРВЕР НА RENDER:
   - Зарегистрируйтесь на render.com
   - Создайте Web Service
   - Загрузите файлы: bot.py и requirements.txt
   - Получите URL вида: https://имя.onrender.com

2. ТЕРМИНАЛ ЭВОТОР:
   - Настройки → Дополнительные → Пользовательский скрипт
   - Вставьте код из loyalty.js
   - Замените apiUrl на ваш URL
   - Сохраните и перезагрузите

3. ТЕЛЕГРАМ БОТ:
   - Откройте @BotFather в Telegram
   - Создайте бота или используйте существующего
   - Получите токен
   - Вставьте в настройки Render
